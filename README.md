Smallpeice Rules
================


`master` is the latest version.

See `releases` or `tags` for individual years.


Requirements
------------

- GNU Make
- Inkscape or `svg2pdf`
- `pdflatex`

How to build
------------

- Clone.
- Initialise Submodules
  - `git submodule init`
  - `git submodule update`
- `make`
